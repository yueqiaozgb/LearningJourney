# Learning Journey

**背景：**

说来惭愧，毕业近一年时间几乎无所寸进，一直在开发和维护小型系统，除了熟练的代码堆砌，对于系统开发的流程已经忘记，导致每每完成的项目都有一些不完美，随着时间的飞逝，越来越感觉自身的能力不足，无论是生活、工作上的琐事，亦或者自身的懒惰性，都导致自身的学习意愿大幅度下降，在我大学时，浪费了太多的时间，现在工作了，又要重蹈覆辙，以至于现在的路走的十分艰难，现在创建这个仓库，用来激励自己重新拾取学习的方向和动力。

**目标：**

- 深化掌握的技能：

  目前的开发，仅仅停留在知道如何使用的地步，看面经的时候，发现大多在问一些底层的实现，这些东西平时并不关注，要加强学习

- 拓展新的技能：

  - 横向发展，学习golang，rust，用不同的语言实现类似的需求
  - 纵向发展，学习大数据开发
  - 新发展，学习AI+其他

**总结：**

从目前了解的信息来看，就业市场很差，有份工作就不错了，为了避免吊死在Java的树上，全栈开发不可避免

目前的路线

- 主要语言：Java
- 主要数据库：Mysql、Redis、MongoDB、Clickhouse
- 消息中间件：RabbitMQ、Kafka
- 运维容器化：docker
- 前端框架：Vue

未来的发展

- AI智能框架：后期
- 大数据框架：中期
- Golang、Rust开发：前期
