# 引言

手撕算法算是面试的基操了，长时间没用接触算法，有一定的生疏，本目录用于记录掌握各类常见数据结构和算法的过程



# 最终目标

- 数据结构

在工作中经常使用列表和哈希表，对于数组和其他数据结构的使用并不多，而leetcode中大多数题目经常使用数组和树，先从数组开始学习

- 算法

算法已经很久没使用了，深度优先，广度优先，回溯，二分，路径规划等等，本次重点需要弥补的是算法部分



# 学习路线

根据网络书籍与leetcode官网，数据结构和算法主要分为以下种类

![image-20241119002739221](C:\Users\yueqi\Desktop\LearningJourney\leetcode\assets\image-20241119002739221.png)

数据结构：

- 字符串
- 数组
- 链表
- 栈
- 队列、双端队列
- 哈希表
- 树：二叉树、二叉搜索树、平衡二叉树
- 其他特殊结构：并查集、前缀和差分、线段树、树状数组、字典树和后缀数组

算法：

- 排序
- 递归与分治
- 二分查找
- BFS、DFS、拓扑排序
- 回溯法
- 动态规划
- 贪心
- 以及一些经典问题



# 长期规划

1. 根据学习路线，先从字符串开始，做一些简单的题目练手，熟悉在不借助代码提示的情况下完成算法
2. 多练习简单题目，参加一些周赛，能够快速写出解
3. 主攻中等难度项目，查看论坛，了解面试难度和方向
4. 归纳并总结，完成数据结构和算法的训练，并保证每周训练量，不至于手生疏
5. 写题解，不仅能够解，还要能够表达
6. 查看相关面试题目，针对性训练
